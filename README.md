#

## config

保存了一些 linux 或其他软件的配置文件

```sh
#批量添加软链
ln -s $HOME/tools/config/* ~/

#linux
ln -s $HOME/tools/config/.tmux.conf ~/.tmux.conf
ln -s $HOME/tools/config/.zshrc ~/.zshrc
#develop tools
ln -s $HOME/tools/config/.vimrc ~/.vimrc
# node
ln -s $HOME/tools/config/.npmrc ~/.npmrc
```
